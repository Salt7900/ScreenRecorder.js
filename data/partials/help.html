Instructions

  • Click 'New recording' in order to start a new Script
    • The first node you click will be the container node and any interaction with it or it's children ill be recorded
    • In order to stop recording when you are finished, click on the plugin icon again
  • Load Recording
    • Click on the Load Recording button in the main menu in order to open a script from your filesystem
    • After you have selected a script, you will be prompted to either play the script or edit it
  • Edit Recording
  • On this screen you will see a table listing out every action the script takes
  • This information is provided in an editable format allowing you to customize the script
  • Click on the 'Configuration Options' in the upper right corner in order to set configurations altering how the script will execute
